# Augment Code ROI Calculator

Interactive ROI calculator for Augment Code's four automation plays:
- Code Review Automation
- Unit Test Automation  
- Build Failure Analyzer
- Interactive (IDE + CLI)

Includes a Summary tab and PDF export.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project → select this repo
3. Click Deploy (no settings to change)

## Run Locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)
